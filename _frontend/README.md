# Pibble Pics

## What is this?
This is a game where you draw a daily photo with your friends, whether its a memory in the past between you guys or just for the crack 

## How do you use it?
1. Login
2. Click draw
3. Draw an image
4. Post it

## What features are available
- Drawing
- Friends
- Liking
- Commenting
- Saving
- Local auto save
- Push Notifications
- PWA
